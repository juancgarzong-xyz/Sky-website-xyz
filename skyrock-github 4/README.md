# Skyrock Studio

Sitio web oficial de **Skyrock Studio** — Estudio de mezcla, postproducción de audio y video podcast en Bogotá.

## Tecnología

Sitio estático en HTML/CSS/JS puro. No requiere build ni dependencias.

## Deploy en GitHub Pages

1. Sube este repositorio a GitHub
2. Ve a **Settings → Pages**
3. En **Source** selecciona la rama `main` y carpeta `/ (root)`
4. Guarda — en 1-2 minutos el sitio estará en `https://tu-usuario.github.io/nombre-repo/`

## Dominio personalizado

Para usar `sky-rock.co`:
1. En **Settings → Pages → Custom domain** escribe `sky-rock.co`
2. Agrega un archivo `CNAME` con el contenido `sky-rock.co` (ya incluido en este repo)
3. Configura el DNS en Cloudflare apuntando a GitHub Pages
